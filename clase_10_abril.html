1 LA GRAN DIFERENCIA DE HOY

Hasta ahora el chatbot respondía, pero al hacer la siguiente pregunta no se acordaba de lo que habías dicho antes. Hoy eso cambia.

¿Cómo funciona la memoria? Cada vez que hablás con la IA, el código guarda lo que dijiste vos y lo que respondió ella en una lista. Antes de mandar el próximo mensaje, le enviamos toda esa lista junto con la nueva pregunta. Así la IA "recuerda" — porque ve toda la conversación de nuevo.

La lista que guarda todo se llama historial. El código la arma solo — vos solo tenés que elegir el personaje y pegar tu API Key.

import requests # --- ¡MODIFICÁ SOLO ESTAS DOS LÍNEAS! --- personaje = "Iron Man" API_KEY = "gsk_PEGA_TU_CLAVE_ACA_DENTRO" # ----------------------------------------- # Todo lo de abajo ya está armado (NO TOCAR) URL = "https://api.groq.com/openai/v1/chat/completions" headers = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"} historial = [] print(f"🤖 Hablás con {personaje}. Esta IA recuerda todo lo que decís.") print("Escribí 'salir' para terminar.\n") while True: mensaje = input("Vos: ") if mensaje.lower() == "salir": print("¡Hasta luego!") break historial.append({"role": "user", "content": f"Actuá como {personaje}. {mensaje}"}) payload = { "model": "llama-3.3-70b-versatile", "temperature": 0.7, "messages": historial } response = requests.post(URL, headers=headers, json=payload) data = response.json() respuesta = data["choices"][0]["message"]["content"] print(f"\n🤖 {personaje}: {respuesta}\n") historial.append({"role": "assistant", "content": respuesta})

💡 Probá esto: Contale algo al principio — tu nombre, algo que te guste. Hacé 3 preguntas normales. Después preguntale: "¿Qué te conté de mí al principio?" ¿Se acuerda?

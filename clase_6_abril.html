1 GLOSARIO NUEVO

Hoy vamos a usar tres conceptos importantes. Leelos con atención:

El Paquete JSON Cuando la IA nos responde, no manda solo el texto. Manda un "paquete" con mucha información: el texto, qué modelo se usó y cuántos tokens se gastaron en total.
Temperature (Temperatura) Es un número entre 0.0 y 2.0 que podemos sumarle al código. Si le ponemos 0.0, la IA es aburrida y exacta. Si le ponemos más de 1.0, se pone muy creativa (y a veces dice locuras).
Alucinación Es cuando la IA no sabe algo pero inventa la respuesta y te la dice con mucha seguridad, como si fuera verdad. Esto pasa porque la IA no busca en internet, solo adivina palabras.

2 ACTIVIDAD 1: MIRANDO EL PAQUETE COMPLETO

Vamos a conectarnos a la IA y pedirle que nos muestre los tokens que gastó.

PASO A PASO:
  1. Abrí tu cuaderno "mi primer ia" en Google Colab.
  2. Copiá el código de acá abajo en una celda nueva.
  3. Reemplazá tu API Key en la línea 4.
  4. Ejecutá el código y mirá lo que aparece al final.
import requests # Configuración API_KEY = "gsk_PEGA_TU_CLAVE_ACA" URL = "https://api.groq.com/openai/v1/chat/completions" headers = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"} payload = { "model": "llama-3.3-70b-versatile", "messages": [{"role": "user", "content": "Escribí un chiste muy corto."}] } # Envío response = requests.post(URL, headers=headers, json=payload) data = response.json() # Mostrar resultados print("--- RESPUESTA DE LA IA ---") print(data["choices"][0]["message"]["content"]) print("\n--- TOKENS GASTADOS ---") print(data["usage"])

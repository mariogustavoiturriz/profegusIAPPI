1 EL DESAFÍO: CREAR TU CONTRASEÑA

Para que la Inteligencia Artificial nos escuche, primero necesita saber quiénes somos. Para eso necesitamos una API Key (una contraseña secreta).

  1. Entrá a console.groq.com desde el navegador de tu notebook.
  2. Iniciá sesión con tu cuenta de Google del colegio.
  3. En el menú, buscá la opción API Keys.
  4. Hacé clic en el botón "Create API Key".
  5. Copiá la clave larguísima (empieza con gsk_...). ¡Tenela a mano que la vamos a usar en el código!

2 TU PRIMER CÓDIGO

Vamos a conectar Python con la Inteligencia Artificial.

PASO A PASO:
  1. Abrí Google Colab y creá un nuevo cuaderno.
  2. Copiá el código de acá abajo y pegalo.
  3. Reemplazá el texto de la línea 4 por tu API Key real.
  4. Tocá el botón de "Play" para ejecutarlo.
# 1. Importamos la herramienta para internet import requests # 2. Pegá tu clave adentro de las comillas API_KEY = "gsk_PEGA_TU_CLAVE_ACA_DENTRO" URL = "https://api.groq.com/openai/v1/chat/completions" # 3. Armamos la configuración headers = { "Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json" } # 4. Elegimos el modelo y le hacemos la pregunta payload = { "model": "llama-3.3-70b-versatile", "messages": [ {"role": "user", "content": "¿Quién es el villano principal de Batman en los cómics?"} ] } # 5. Enviamos y mostramos la respuesta response = requests.post(URL, headers=headers, json=payload) data = response.json() print(data["choices"][0]["message"]["content"])

Ahora vamos a hacer que la IA actúe como un personaje. Para que no reniegues escribiendo, el código ya está casi todo armado. ¡Vos solo tenés que cambiar los textos del principio!

  2. Copiá este nuevo código y pegalo ahí.
  3. Modificá SOLO las primeras 3 líneas (Escribí qué personaje querés, qué tema querés preguntarle, y pegá tu API Key de nuevo).

import requests # --- ¡MODIFICÁ SOLO ESTAS TRES LÍNEAS! --- personaje = "Iron Man" tema = "qué pensás de la Inteligencia Artificial" API_KEY = "gsk_PEGA_TU_CLAVE_ACA_DENTRO" # ---------------------------------------- # Armamos la pregunta automática (NO TOCAR) pregunta = f"Actuá como {personaje} y explicame {tema}." # Preparamos la conexión y el mensaje (NO TOCAR) URL = "https://api.groq.com/openai/v1/chat/completions" headers = {"Authorization": f"Bearer {API_KEY}", "Content-Type": "application/json"} payload = { "model": "llama-3.3-70b-versatile", "messages": [{"role": "user", "content": pregunta}] } # Enviamos y vemos el resultado (NO TOCAR) response = requests.post(URL, headers=headers, json=payload) data = response.json() print("\n🤖 RESPUESTA DE LA IA:") print(data["choices"][0]["message"]["content"])

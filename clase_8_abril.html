2 EL ESCUDO PROTECTOR

¿Qué es un Escudo Protector? La IA sabe de todo, pero nosotros ponemos las reglas. Un escudo es una condición (un if) que revisa lo que el usuario escribió ANTES de mandarlo a internet. Si el usuario escribe una mala palabra o un tema que no querés, el programa lo frena.
🛑 REGLA DE ORO: ¡NO BORRES NADA!

Para no perder tu trabajo anterior:

  1. Debajo del Reporte QA que acabas de hacer, agregá una CELDA DE CÓDIGO NUEVA (+ Código).
  2. Pegá este nuevo código en esa celda vacía.
  3. Pegá tu API Key en la línea 4.
import requests # ── 1.
